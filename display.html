DISPLAY
Concept Description Graphic
The display property let us control the layout. It specifies how an element is displayed.
By default, the display property is set depending the type of the element.

INLINE

Technicality:
An in-line element does not start on a new line and only takes up as much width as necessary.

How I see it:
It's an individual who only takes his own personal space.

This is an inline element which by default, only takes the space necessary to display.

Example of inline elements:
span, a, img

BLOCK

Technicality:
A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).

How I see it:
This individual takes up a full row and adds a line-break after itself.

Next is an example of a block level element, a paragraph (p)

Which takes full width of the parent container.

Examples of block-level elements:
div, h1-h6, p, form, header, footer, section

INLINE-BLOCK

This property allows to set a width & height on the element.

Top and bottom margins/paddings can be set unlike inline elements.

Does NOT add a line-break after the element.
