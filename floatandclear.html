FLOAT & CLEAR
Concept Description Graphic

FLOAT

Technicality: The Margin is the specified area outside the box's border.

How I see it: It's the personal space the box takes from other elements to avoid touching them *COVID free*

FOUR VALUES SYNTAX:
Margin: top right bottom left;

THREE VALUES SYNTAX:
Margin: top leftright bottom;

TWO VALUES SYNTAX:
Margin: topbottom leftright;

CLEAR

Technicality: The Padding is the specified area between the content and the border.

How I see it: It's the amount of clothing layers the content (Joey´s body) has.

FOUR VALUES SYNTAX:
Padding: top right bottom left;

THREE VALUES SYNTAX:
Padding: top leftright bottom;

TWO VALUES SYNTAX:
Padding: topbottom leftright;
