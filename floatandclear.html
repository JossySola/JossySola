FLOAT & CLEAR
Concept Description Graphic

FLOAT

This property is used for positioning and formatting content. Specifies how an element should float.

My common sense says that in order to make something float, it has to be an inline/inline-block level element that does not occupy a full width.
Float left Float right

This property can have one of the following values: left, right, none, inherit.

CLEAR

This property specifies what should happen with an element that is next to a floating element.
Float left Float left

Float left Clear left

The possible used values are: none, left, right, both, inherit.
